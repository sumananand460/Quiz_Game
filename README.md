# 🎯 Quiz Game Application
## Project LINK: https://sumananand460.github.io/Quiz_Game/

A fully interactive and responsive Quiz Game built using **HTML, CSS, and Vanilla JavaScript**.

This project demonstrates dynamic DOM manipulation, state management, event handling, and user interface updates without using any external libraries or frameworks.

---

## 🚀 Features

✔ Start screen, quiz screen, and result screen (Single Page Layout)  
✔ Dynamic question and answer rendering  
✔ Real-time score tracking  
✔ Progress bar indicator  
✔ Answer feedback (Correct / Incorrect highlighting)  
✔ Result evaluation based on percentage  
✔ Restart functionality  
✔ Fully responsive design  

---

## 🛠 Technologies Used

- **HTML5** – Structure and layout  
- **CSS3** – Styling, animations, responsive design  
- **Vanilla JavaScript (ES6)** – Logic and interactivity  

No frameworks or external libraries were used.

---

## 🧠 Core Concepts Implemented

This project demonstrates understanding of:

- DOM manipulation
- Event listeners
- Dynamic element creation
- JavaScript state management
- Conditional rendering
- setTimeout for delayed transitions
- Dataset property usage
- Array methods (forEach, Array.from)
- Responsive UI design
